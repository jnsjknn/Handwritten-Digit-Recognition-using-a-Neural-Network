Simply draw a digit on the canvas with your mouse or finger!

The neural network tries to guess which digit you drew.

For example, if you see this:


the neural network thinks you most likely drew a 1 and possibly a 7.

This app uses a pure JavaScript neural network to recognize hand written digits. To train the neural network I collected a dataset of rougly 14000 handwritten digits. The digits were draw by more than 200 volunteers.

You can help make the dataset larger. For more information about the project, take a look at this project on Github.
